This is a github tutorial
1- Initialize a repository
>> git init
2- Add changes to staging directory
>> git add .
3- Commit the changes (make that new node)
>> git commit -m"Your message here"





Helpful Commands for Junsung
Shows the status of what has been changed/commited
>> git status
Shows all your commits in the past
>> git log
To go back to see old versions copy the shaw value from git log
>> git checkout #shaw
